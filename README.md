# GOSH

```
Simple static file server with no dependencies                     

Usage:                           
  gosh [options]                 

Options:                         
  -addr string                   
        binding address (default "127.0.0.1:3000")                 
  -dir string                    
        serve files from directory (default ".")                   
  -help                          
        show this help
```
